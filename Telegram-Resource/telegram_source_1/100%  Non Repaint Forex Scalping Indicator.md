# بررسیِ منبع: `100%  Non Repaint Forex Scalping Indicator/` (Binary Code Indicator)

> قانونِ شمارهٔ ۱: تابعِ هدف = **سودِ خالصِ بیشتر** (XAUUSD + EURUSD)؛ WR کفِ ۴۰٪ دارد.
> مشخصاتِ حساب: `CONTRACT_SIZE=100`، طلا `3.3pip`، کمیسیون=صفر.

شانزدهمین موردِ بررسی‌شده (ترتیبِ الفباییِ `LC_ALL=C`).

## ۱. چیستی

| ویژگی | مقدار |
|---|---|
| **نوع** | سیستمِ تمپلیت‌محورِ MT4 با اندیکاتورِ **کامپایل‌شده (`.ex4`)** — سورس بسته. |
| **اجزا** | `Binary Code Indicator.ex4` (arrowِ سیگنال) + `Timer-Best.ex4` (شمارشِ زمان) + تمپلیتِ `Forex Binary Code.tpl`. |
| **تایم‌فریمِ تمپلیت** | **`period=60` (H1)**. |
| **ادعا** | «100% No Repaint Scalping» (تبلیغاتی؛ به‌ویژه برای Binary Options طراحی شده). |

## ۲. منطقِ استخراج‌شده

- منطقِ عددی از `.ex4` قابلِ استخراج نیست (سورس بسته).
- **قرینهٔ مهمِ ضدِ ادعا:** در تمپلیت، سیگنال‌های `Binary Code up/dn` به‌صورتِ
  **object با timestampِ ثابت** ذخیره شده‌اند (مثلاً `Binary Code up 2019.07.29 03:00`).
  این یعنی arrowها **پس از وقوع** روی نقاطِ «درست» نشسته‌اند — نشانهٔ کلاسیکِ اندیکاتورهای
  **repaint** (برخلافِ ادعای «No Repaint») یا arrowهای بصریِ ثابت. صحتِ ادعا رد می‌شود.
- طراحیِ اصلی برای **Binary Options** است (پیش‌بینیِ جهتِ کندلِ بعد)، نه اسکلپِ فارکسِ TP/SL‌دار.
